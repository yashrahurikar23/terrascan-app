## Step 1: Understand Your Goal

You're building a **GDAL Image Processing App** using Streamlit. This app will:
- Process geospatial images
- Extract metadata and statistics
- Perform image operations
- Calculate spectral indices (like NDVI)
- Create visualizations

---

## Step 2: Start with Fundamentals

**Begin here:** `01_fundamentals/README.md`

**What you'll learn:**
- What digital images are
- How computers represent images
- Basic image properties

**Time:** 2-3 hours

**Action Items:**
1. Read the README
2. Complete the exercises
3. Try opening an image in Python
4. Calculate basic statistics

---

## Step 3: Learn GDAL Basics

**Next:** `02_gdal_basics/README.md`

**What you'll learn:**
- How to use GDAL
- Opening images with GDAL
- Reading band data
- Extracting metadata

**Time:** 3-4 hours

**Action Items:**
1. Read the README
2. Complete the exercises
3. Try opening a GeoTIFF with GDAL
4. Extract image properties
